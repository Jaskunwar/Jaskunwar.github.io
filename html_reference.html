Html Tages

  1. <>:This is where put your html tages
  2. !DOCTYPE: It defines the type of document
  3. body: Where you have a paragraph or the body of your website
  4. h1-h6 : Used to write heading. The h1 one being the biggest font nad h6 being the smallest font.
  5. head: It defines the information about the content in the document, and it is also another place to type title.
  6. br: It is used to add a line break
  7. div: It defines the different types of document
  8. style: It add a visual change to the content eg: the color or the type of font used
  9. title: It defines the title for the website
  10. ul: It defines an unordered list eg: bulletin points
